  # 划词翻译器

  > 告别截图翻译和软件切换 —— 选中文本，一键翻译，适用于 Windows 所有应用。

  ## 这是什么？

  划词翻译器是一个轻量级的 Windows 桌面翻译工具。在任何软件中选中英文文本，按下 `Alt+Z`，翻译结果就会以浮动窗口的形式出现在鼠标旁边。

  无需打开浏览器、无需复制粘贴、无需切换窗口。翻译过程调用免费的在线翻译服务，不消耗任何 AI Token，完全免费。

  ## 效果演示

  1. 在浏览器 / PDF / Word / 代码编辑器等任意软件中选中英文文本
  2. 按下 `Alt+Z`
  3. 鼠标旁弹出翻译结果，阅读后点击任意位置或按 `Esc` 关闭

  ## 安装

  ```bash
  git clone https://github.com/houdeTed/-Word-Translator-.git
  cd -Word-Translator-
  pip install -r requirements.txt

  使用

  python main.py

  启动后在系统托盘静默运行。选中英文文本 → 按 Alt+Z → 弹窗显示翻译。

  翻译服务

  按顺序尝试以下免费服务，一个不可用则自动切换到下一个：

  1. 有道翻译
  2. 彩云小译
  3. 爱词霸
  4. 阿里巴巴翻译

  无需注册，无需 API Key。默认英译中，可修改 translator.py 切换语言方向。

  常见问题

  Q: 按下 Alt+Z 没有反应？
  A: 快捷键可能被其他软件（截图工具、QQ、微信等）占用，修改 hotkey.py 更换快捷键即可。

  Q: 翻译失败怎么办？
  A: 程序会自动尝试多家翻译服务，全部失败请检查网络连接或提 Issue。

  Q: 支持其他语言吗？
  A: 支持。编辑 translator.py 中的 source 和 target 参数即可。
  ```
